◆【Picture Help】

左・フレーム が メニュー になります，その メニュー の アイコン を クリック すると右・フレーム に その画像が表示されます．
”Museum”の右・フレーム の ”ビュー・エリア”（フレーム・内の｛画像 又は，緑色の エリア｝）内で マウス・カーソル を上部・中央あたりへ動かすと， ”ビュー・エリア”の上部・中央から Zoom メニュー が表示されますので，その Zoom メニュー の の ボタン・アイコン や「 ”S”から最大で ”3L”まで」の文字を マウス で クリック すると サイズ が変更されます．
たまに 画像が表示されなかったり Zoom メニュー が出ない事も有りますが，まあ それは ご愛嬌と言う事で…(^_^;)
