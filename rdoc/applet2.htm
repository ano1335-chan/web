Applet に関する ヘルプ

●《イメージ「Applet」ページ に関する ヘルプ》
※ Applet は他の アプリケーション と相性（あいしょう）が悪い場合が有りますので，Applet の有る ページ を開く前（まえ）に なるべく不要な アプリケーション は閉じて下さい．
又，もし Applet が通常（つうじょう）よりも小さく表示された場合は ブラウザ の再表示を実行して下さい．
※この ”イメージ 「Applet」”の ページ を開くと左・フレーム に その タイトル と Applet・アイコン が表示されます．
その タイトル を マウス で クリック すると 右・フレーム に その Applet の ページ が開きます，又 左・フレーム の Applet・アイコン を マウス で クリック すると その ページ・内 の Applet の場所が表示されます．
●《Applet・操作に関する ヘルプ》
※｛ Picture；Slide；Movie ｝の Applet の画像が正常に表示できない場合は Applet の画像・表示・エリア を マウス で クッリック すれば画像の再・ロード が実行され，その｛ Slide；Movie ｝が再生・表示中の場合 その操作によって再生・表示は停止し， ”Pause”ボタン は ”Play”ボタン に変化します．
それでも Applet が正常に表示されない場合は下記の順に ためしてみて下さい．
（＃１）． 左・フレーム から右・フレーム に別の Applet の ページ を開いてから元の Applet の ページ を開きなおすか，又は 左・フレーム の タイトル や Applet・アイコン から Applet の ページ を別・ウィンドウ で開く．
なお， ”Windows”上（じょう）での別・ウィンドウ の開きかたは下記の『”Windows”の ブラウザ に関する操作・ヘルプ』を参照して下さい．
（＃２）．いったん その ブラウザ によって開かれている全ての ウィンドウ を閉じてから この ホーム・ページ を表示しなおす．
（＃３）．コンピュータ を再起動して この ホーム・ページ を表示しなおす．
※｛ Slide；Movie ｝の Applet は右端の ボタン を マウス で クリック すると次の画像を表示し，最後の画像の場合は１番目の画像に戻（もど）ります，又 左端の ボタン を マウス で クリック すると１つ前（まえ）の画像を表示し，１番目の画像の場合は最後の画像が表示されます．
Applet が再生・表示中の場合は それらの操作によって再生・表示は停止し， ”Pause”ボタン は ”Play”ボタン に変化します．
△｛ Slide；Movie ｝の Applet スクロール・バー を マウス で ドラッグ して移動させると それに ともない画像も変化し，その Applet が再生・表示中の場合は再生・表示は停止し， ”Pause”ボタン は ”Play”ボタン に変化します．
△｛ Slide；Movie ｝の Applet の ”Play”ボタンを マウス で クリック すると画像は順次（じゅんじ） 再生・表示され， ”Play”ボタン は ”Pause”ボタン に変化します．
△｛ Slide；Movie ｝の Applet の再生・表示中に ”Pause”ボタン を マウス で クリック すると再生・表示は停止し， ”Pause”ボタン は ”Play”ボタン に変化します．
又 この｛ Slide；Movie ｝の Applet の再生・表示が終了した場合は ”Pause”ボタン が ”Play”ボタン に変化します．
なお Applet・ウィンドウ の左上に表示されている ”Slide”の カッコ の中の数字の意味は（現在の表示・枚数／最大・枚数）と なっています．
★右・フレーム の ページ・内（ない）に有る リンク を別・ウィンドウ で開けば現在の ページ と比較しながら読む事が できます．
その右・フレーム の ページ・内（ない）に有る リンク に ジャンプ する場合に少々 タイム・ラグ が生（しょう）じる事も有るので ご注意 下さい．
なお， ”Windows”上（じょう）での別・ウィンドウ の開きかたは次の『”Windows”の ブラウザ に関する操作・ヘルプ』を参照して下さい．
