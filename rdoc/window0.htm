●《「 ”Internet Explorer”，”Netscape Communicator(Navigator)”」に関する操作・ヘルプ》
◆《検索に関する操作・ヘルプ》
☆検索の漢字・入力に関する注意．
検索・ダイアログ に漢字・入力できない場合は 一旦（いったん） ブラウザ を閉じて下さい．
それでも漢字が入力できない場合は日本語・IME の ”入力・モード”を マウス の左・クリック で メニュー を開き ”ひらがな”を選択して下さい．
なお，この日本語・IME が半角・英数字・入力の場合は ”入力・モード”は ”A”と表示されている はずです．
☆ ”Internet Explorer ５”による フレーム・表示・状態での検索に関する操作・ヘルプ．
必ず検索・ダイアログ を開く前（まえ）に検索したい フレーム の エリア・内（スクロール・バー も含（ふく）む）を マウス の左・クリック によって（検索する フレーム を）アクティブ に してから検索して下さい．
又 別の ページ を開いた時に すでに検索・ダイアログ が開ている場合も，検索・ダイアログ を閉じてから検索したい フレーム の エリア・内（スクロール・バー も含（ふく）む）を マウス の左・クリック によって（検索する フレーム を）アクティブ に してから検索して下さい．
☆ ”Netscape Communicator(Navigator) ４．７５”による フレーム・表示・状態での検索に関する操作・ヘルプ．
検索・ダイアログ を開く前（まえ）に検索したい フレーム の エリア・内（スクロール・バー も含（ふく）む）を「マウス の左・クリック 又は マウス・ホイール の スクロール」によって（検索する フレーム を）アクティブ に してから検索して下さい．
又 別の ページ を開いた時に すでに検索・ダイアログ が開ている場合は，検索・ダイアログ を閉じずに検索したい フレーム の エリア・内（スクロール・バー も含（ふく）む）を「マウス の左・クリック 又は マウス・ホイール の スクロール」によって（検索する フレーム を）アクティブ に すれば検索できます．
◆《リンク に関する操作・ヘルプ》
☆同一・ウィンドウ・表示の リンク で新たに 別・ウィンドウ を開く場合の操作・ヘルプ．
△リンク に対し マウス の右・クリック によって ショートカット・メニュー を開き，その［新しい ウィンドウ を開く］に マウスの左・クリック を おこなっう事によって別・ウィンドウ を開く事ができます．
又 その ウィンドウ が不要になった時には，その ウィンドウ を閉じて下さい．
別・ウィンドウ を開く事が できない場合は 一旦（いったん） ブラウザ を閉じるか 又は コンピュータ を再起動して下さい．
